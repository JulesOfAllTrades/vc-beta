Using Repl.It Version Control

Get repl.it ready

  1. Sign up for a Repl.It account
  2. Click the plus sign in the upper right corner to create a new repl.
  3. Use the version control option in the left nav to connect your GitHub account.
  4. Add a new repo directly from repl.it.
  5. Add a readme.md and index.html file.

Get GitHub ready

  1. Open GitHub desktop app and clone the repo.
  2. Back in the GitHub brower, add a folder to the repo.
  3. Add a copy of your index.html file to the folder.
  4. In the repo settings, select Pages.
  5. Then select your main branch and the root folder.
  6. This should generate a link for your github pages site.

Back to repl

  1. In the version control menu, select the option to pull.
  2. Add content to your index.html files.
  3. Commit your changes (push to GitHub),
