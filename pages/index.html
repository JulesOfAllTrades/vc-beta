Using Repl.It Version Control

Get repl.it ready

  1. Sign up for a Repl.It account
  2. Click the plus sign in the upper right corner to create a new repl.

Git goin' with GitHub

  1. Sign up for a GitHub account
  2. Click the plus sign in the upper left corner to create a new repo.
  3. Name your repo. I called this one repl-version-control-info.
  4. Configure as needed, I just made this public and added a readme file and description.

All together now

  1. If it is available to you, select the version control icon from the left nav on your repl.
  2. Connect to GitHub and log in.
  3. Select the existing repo you recently created.
  4. Add a folder to your repl.
  5. Add an index.html file to your repl in the folder you created.

Commit

  1. In your repl, in the version control menu, add a commit message.
  2. Then hit commit and push.
  3. Go to GitHub to see your new file!
